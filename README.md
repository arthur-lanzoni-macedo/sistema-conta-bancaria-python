💳 Sistema de Conta Bancária em Python
======================================

Projeto simples desenvolvido em **Python** com o objetivo de praticar **Programação Orientada a Objetos (POO)**.

O sistema simula operações básicas de uma conta bancária, permitindo criar contas, realizar depósitos, saques e transferências entre contas.

🚀 Funcionalidades
------------------

*   Criar contas bancárias
    
*   Realizar depósitos
    
*   Realizar saques
    
*   Transferir dinheiro entre contas
    
*   Exibir dados da conta
    

🧠 Conceitos aplicados
----------------------

*   Programação Orientada a Objetos (POO)
    
*   Classes e Objetos
    
*   Métodos
    
*   Atributos
    
*   Manipulação de dados entre objetos
    

🛠️ Tecnologias utilizadas
--------------------------

*   **Python 3**
    

📌 Exemplo de uso
-----------------

```
conta1 = ContaBancaria('Arthur', 1000, '001')
conta2 = ContaBancaria('Maria', 500, '002')


conta1.depositar(300)
conta1.sacar(200)
conta1.transferir(400, conta2)


conta1.mostrar_dados()
conta2.mostrar_dados()
```
